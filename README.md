# mac tool-kit

My first vibe coding project.

## Purpose

I want use this project to replace those third-party tools.

## Current features

- Select a screen region and capture it directly from the overlay.
- Adjust the selected region before capture with resize handles, a size label, and a bottom action toolbar.
- Move the selected region by dragging inside it before confirming capture.
- Annotate the selected region with rectangle, arrow, pencil, marker, a combined privacy brush with blur/mosaic options, text, undo, redo, delete, and selected-annotation color controls.
- Select, move, resize, delete, and double-click text annotations before capture/export.
- Capture/copy the selected region with the toolbar checkmark, Return, or double-click inside the selection.
- Pin the selected region from the toolbar without closing the clip overlay; pinned snapshots keep screen-scale resolution, can move while clipping continues, show controls on hover/selection, support Escape close when selected, keyboard/pinch zoom, and context menu copy/save/OCR/close.
- Run OCR from the separate OCR toolbar action and keep the clip overlay open.
- Recognize clipped screen content locally with English, Simplified Chinese, and Traditional Chinese recognition.
- Format JSON with two independent JSON editor panes.
- Automatically log text and image copies on this Mac while clipboard monitoring is enabled; search and filter persistent copy history, inspect full text or image previews with metadata, copy/delete entries, and load JSON history into either JSON editor.
- Generate, decode, copy, and preview upcoming run times for standard 5-field cron expressions.
- Compare two text blocks side-by-side with line-level diff results and highlighted changed text.
- Use Settings to control launch at login, hide-to-menu-bar behavior, default clip save location, clipboard monitoring, history limits, image history size, and OCR language mode.
- Use the macOS menu bar icon to open ToolKit, start screen clipping, jump to Copy History or Settings, pause/resume clipboard monitoring, or quit explicitly.
